API to offer OpenGraph meta or oEmbed media.

> OpenGraph example: <https://social-oembed.git-projects.xyz/?url=https://github.com>

## **Setup**

Download dependencies

```bash
bun i
```

Create `.env`

```bash
cp .env.example .env
```

Execute `start`

```bash
bun start
```
